## 🤔 如何参与

我们欢迎各种形式的参与！

### 提出新问题
如果你在研究过程中遇到：
- 数学推导中的困惑
- 实验现象的疑问（为什么有些时候一个现象在另一种情况下不成立）  
- 论文中未充分解释的点（有一些论文可能具有很好的实验结果，但是并未做出详细的理论分析和证明）

请创建 [New Issue](https://github.com/Shiying-Zhang/diffusion-theory-discussion/issues/new/choose) 或直接添加问题文档。

### 贡献解答
- 完善现有问题的解答
- 添加新的理论分析
- 修正数学推导错误
- 补充参考文献
- 使用实验证明假说

### 参与讨论
在 Issue 中评论，或在文档中提出你的见解。
